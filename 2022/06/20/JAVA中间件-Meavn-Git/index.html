JAVA中间件-Meavn&Git

Meavn

  • 作用
  1. 相同的项目结构—-pom.xml
  2. 统一维护jar包—-一套jar仓库，配置本项目所需的jar版本即可
  • 下载
  1. 官方最新版下载地址：http://maven.apache.org/download.cgi
  2. 配置环境变量
  • 仓库
  1. maven采用一个仓库，多个项目的方式，让多个项目共享一个仓库里的相同jar包。
  2. 位置：settings.xml中配置

GIT

  • 作用
  1. 项目的版本控制
  2. 多人协作
  3. http://github.com
  • 下载安装
  1. 地址：https://git-scm.com/download/win
  2. 检查：git –version 显示版本安装成功
  • 使用
  1. idea配置使用git：菜单->settings->Version Control->Git->Path to Git executable: 设置为安装git中所安装的git.exe
  2. idea设置GitHub账号密码：菜单->settings->Version Control->GitHub->Create API Token
  3. checkout下拉（pull）代码：菜单->VCS->Chekout from Version Control->GitHub
  • 创建git仓库
  1. github上创建仓库获得仓库地址
  2. 菜单->VCS->import into Version Control->Create Git Repository->”你的项目地址”
  3. 把项目加入到本地仓库：右键项目->Git->Add
  4. 提交：右键项目->Git->Commit Directory
  5. Push Commit：填入GitHub获得的仓库地址
