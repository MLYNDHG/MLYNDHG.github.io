JAVA中间件-Redis

Redis

  • 下载安装

  1. JDK版本：至少使用JDK8版本

  2. 什么是Redis：

    基于内存亦可持久化的日志型 Key-Value数据库 Redis就像是一个HashMap 不在JVM中运行以独立进程的形式运行 当作缓存使用 比数据库(mysql)快，常用的数据放在这里可以提高性能

  3. 下载：http://redis.io windows版本的下载地址是： http://redis.io/download

  • 使用

  1. 服务器开启

    redis-server.exe

  2. 客户端访问

    redis-cli.exe

  • 常见命令
  1. ​ String（字符串）：set get incr type ttl rename
  2. ​ List（列表）：lpush rpush llen lrange lpop rpop
  3. ​ Hash（字典）：hset hgetall hkeys hvals
  4. ​ Set（集合）:SADD SMEMBERS
  5. ​ Sorted Set（有序集合）:zadd zcard zcount
  6. ​ 命令手册http://www.redis.cn/commands.html
  • Jedis
  1. 是什么：在java项目代码中使用redis，不仅仅是redis给的客户端中，只需引入jar包即可,即jedis.jar
  2. 使用：javaProject中可直接当作客户端来操作redis，但是redis服务端还是要开启（不然呢？？？）
  • 集成Spring
  1. 开启redis服务端
  2. spring配置连接池，连接工厂，注入RedisTemplate ，getBean即可
  • 可视化redis
  1. 工具redisClient可查看当前连接池的所有数据
