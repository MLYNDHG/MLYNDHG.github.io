------

1. 列表 (List)：如何存储一系列数据（例如 `[ '番茄', '玉米', '黄瓜' ]`）。

   eg:vagetables = [ '番茄', '玉米', '黄瓜' ];

2. `if-elif-else` 条件语句：如何让程序根据不同情况做出不同选择（例如：`if 天气 == '下雨': 就不浇水`）。

3. `for` 循环：如何让程序重复执行任务（例如：检查*每一块*土地）。

   vegetables = [ '番茄', '玉米', '黄瓜' ];

   for veg in vegetables:

     print("我喜欢吃" + veg)

4. `while` 循环：如何让程序在某个条件满足时持续执行。
